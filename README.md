# VerifyMe
VerifyMe by Linxing Huang
<br>
A simple, opensource Authenticator app.
<br>

## Features
- QR code scanning
- Private Protection
- Click Quick Copy
- Backup and restore in json
- Monet Color
- Android 12+ Material Design
