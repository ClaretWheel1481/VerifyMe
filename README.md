# VerifyMe

VerifyMe by Linxing Huang
<br>
A simple, opensource Authenticator app.
<br>

## Features

- QR Code Scanning
- i18n Support
- Private Protection
- Click Quick Copy
- Backup and Restore in Local JSON file or WebDAV
- Monet Color
- Android 12+ Material Design
