# VerifyMe
VerifyMe by Linxing Huang
<br>
一个简单、开源的双重验证码生成器。
<br>

## 特性
- 二维码扫描输入
- 隐私保护
- 单击快速复制
- 通过Json文件备份和恢复
- Monet取色
- Material Design
